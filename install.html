    Q: Is it customizable (if so, how do you show this?)?

    Yes. You can add custom colors to your posts (entire RGB range). You can also edit the code and change/add your personal color definitions.

    Default Colors

  • - red
  • - pink
  • - orange
  • - yellow
  • - green
  • - blue
  • - purple
  • - brown
  • - black
  • - white
  • - undefined

    • To add more colors, place this code in the JS:

    $('body').on('click','#yourColor-info', function(){ $('#yourColor').toggle(); })

    To add more colors, place this code in the HTML (remove spaces after the < sign):

    ${postAndArchive.post.colors} var postContent = { "title" : formTitle, "timestamp" : timestamp, "content" : formContents, "colors" : formColors } < a href="javascript:void(0)" class="color-info" id="red-info">Red < span id="arrow">& #9660; < div class="color-ref" id="red" style="display: none;"> colorContainer.insertAdjacentHTML("beforeend", `
    < li id="colorList"> < a href="./posts/post-${ postAndArchive.post.timestamp}.json" id="hexColorLink" class=${showColors} style="color:${postAndArchive.post.colors};">${postAndArchive.post.colors} `)


    postContainer.insertAdjacentHTML( "beforeend", `
    < h2>$ {postAndArchive.post.title} < h4 id="timestamp">$ {postAndArchive.post.timestamp}

    $ { postAndArchive.post.content}

    < p id="hexColor" class=${showColors} style="color:$ {postAndArchive.post.colors};">$ {postAndArchive.post.colors}




    `)     let showColors = postAndArchive.post.colors == undefined ? "hidden" : "colors"
